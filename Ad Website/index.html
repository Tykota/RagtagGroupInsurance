Simple Interface

We use a web interface in order to manage clients that are using our mobile application. This creates a very user friendly experience on both sides.

Android Application

Our Android application is the user side of the experience. The user can do a number of things. They can submit an application for review to get insurance, view their insurance card, submit a claim. There are features like, crash detection and location tracking when submitting a claim.

star

Web Manager

The web manager is able to review applications, accepting or denying them. The manager may also review any claims that have been made and the location in which the incident happened, as well as search the Firebase finding a specific user.

Usage

The usage for our project is for a car insurance company to be able to have a very easy time interacting with their clients, while giving the clients a user friendly interface to be able to manage their insurance and submit a claim.The Android application to go along with the project can detect crashes as well as give location data for the insurance company to get a location of which a claim has been submitted. The company gets a page that is connected to the database of users where they can manage the users data, as well as manage any claims and accept or deny their policies.
